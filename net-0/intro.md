# Commands

## Run all virtual machines in same time
```bash
    lstart -o "--con0=-xterm" -d -net0
```

## Close all virtual machines
```bash
    lhalt
```

## Restart all running virtual machines
```bash
    lcrash
```

## More commands:
```bash
    man netkit
```
